# PROYECTO UBICACION GPS🧳🌍  

Un proyecto desarrollado en **React** para la gestión de turistas y destinos.  

## 📌 Características  
- Brinda 2 rutas pra tu ubicacion.  
- Listado de destinos.  
- Interfaz intuitiva y amigable.  
- Integración con API externas.
